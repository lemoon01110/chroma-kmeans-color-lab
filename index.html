A computer finds patterns · by itself

Chroma Lab

A photo is made of tiny colored dots called pixels. This site shows how a computer program called K-Means picks a small set of colors that still look like the picture — by guessing, grouping, and fixing its guess over and over.

100% on-device

The big idea (watch the movie →)

Imagine dumping all the pixels from a photo onto a table as colored dots. Similar colors land near each other and make clumps. K-Means tries to put a flag in the middle of each clump.

It does not know the “right” answer. It starts with a wild guess, then runs a short loop until the flags stop moving. Those final flag colors are the palette.

1

Guess. Drop K flags in random places. First try is usually messy — that’s OK.

repeat this loop
2

Group (Assign). Every dot joins the closest flag. Now you have K teams of colors.

3

Move (Update). Slide each flag to the middle of its team — like finding the average color of that group.

↻ keep going while the flags are still sliding a lot
4

Done (Converge). When flags barely move, stop. Paint every pixel with its team’s flag color. That’s the simplified picture.

More detail — if you want it

optional
Every screen color is three numbers: how much Red, Green, and Blue (0 to 255). If you pretend those are left/right, up/down, and depth, each pixel is a spot in space. A photo is a huge cloud of spots. K-Means plants K flags in that cloud and gives every spot to the nearest flag.
1

Guess. Put K flags down (random, or a smarter spread called k-means++).

2

Group. For each pixel, find the closest flag and join that team.

3

Move. Put each flag in the “middle” of its team (the average color). Empty teams get a new random flag.

4

Done. Repeat Group → Move until the flags almost stop sliding.

Last step: repaint every pixel with its team’s flag color. Now the picture only uses K colors.

Nobody told the computer “this is sky” or “this is a leaf.” It only used a closeness rule. Grown-ups call that unsupervised learning — learning without labels. Start from a different random Guess and you might get a slightly different palette. That’s normal!

Experiments
Quick tries
How we measure “near”
RGB uses the three screen numbers as-is — simple. LAB is a different ruler that matches human eyes a bit better, so teams often look nicer.
Where the first flags land
Random = throw flags anywhere (wild early steps). k-means++ = spread the first flags apart on purpose (usually calmer results).
Try a new Guess
Same photo, same K, but brand-new random flag starts. If the colors change, you’ve seen that there isn’t only one “right” answer.
